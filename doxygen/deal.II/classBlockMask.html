Detailed Description

该类代表一个掩码，可用于选择有限元的单个矢量块（另见 该术语条目 ）。它通常有和有限元块一样多的元素，人们可以使用 operator[] 来查询某个特定块是否被选中。 该类的语义与相关的ComponentMask类相同，即一个默认构造的掩码代表所有可能的块。关于这些语义的更多信息见那里。 这类对象在很多地方被使用，人们希望将操作限制在某个块的子集上，例如在 DoFTools::extract_dofs. 中，这些对象可以手工创建，或者更简单，要求有限元从某些选定的块中生成一个块掩码，使用这样的代码，我们创建一个掩码，只表示一个斯托克斯元的速度块（见 Handling vector valued problems ）。

// Q2 element for the velocities, Q1 element for the pressure
FESystem<dim> stokes_fe (FESystem<dim>(FE_Q<dim>(2), dim), 1,
FE_Q<dim>(1), 1);
FEValuesExtractors::Scalar pressure(dim);
BlockMask pressure_mask = stokes_fe.block_mask (pressure);

注意，通过将速度元素包装成一个FESystem对象，我们确保整个元素只有两个块。其结果是一个块掩码，在2D和3D中都有[false, true]的值。(将其与ComponentMask文档中讨论的相应组件掩码进行比较)。同样地，使用

FEValuesExtractors::Vector velocities(0);
BlockMask velocity_mask = stokes_fe.block_mask (velocities);

将导致2d和3d中的掩码 [true, false] 。

unsigned int BlockMask::size ( ) const

如果此块掩码已被初始化为大小大于0的掩码，那么返回此对象所代表的掩码的大小。另一方面，如果这个掩码已被初始化为一个空对象，代表一个对每个元素都是真的掩码（即，如果这个对象在调用 represents_the_all_selected_mask()时将返回true），那么返回0，因为没有明确的大小。

◆ operator[]()

bool BlockMask::operator[] ( const unsigned int  block_index) const

返回一个特定的块是否被这个掩码所选择。如果这个掩码代表了选择all blocks的对象的情况（例如，如果它是使用默认的构造函数创建的，或者是从bool类型的空向量转换而来的），那么无论给定的参数是什么，这个函数都返回true。

Parameters
block_index该函数应返回是否选择该块的索引。如果这个对象代表一个掩码，其中所有的块总是被选中，那么这里允许任何索引。否则，给定的索引需要在零和这个掩码所代表的块数之间。

◆ represents_n_blocks()

bool BlockMask::represents_n_blocks ( const unsigned int  n) const

返回此块掩码是否正好代表 n 块的掩码。如果它被初始化为一个正好有 n entries of type bool 的向量（在这种情况下， n 必须等于size()的结果），或者它被初始化为一个空向量（或使用默认构造函数），在这种情况下，它可以代表一个有任意数量块的掩码，并且将总是说一个块被选中，这就是真的。

◆ n_selected_blocks()

unsigned int BlockMask::n_selected_blocks ( const unsigned int  overall_number_of_blocks = numbers::invalid_unsigned_int) const

返回这个掩码所选择的区块的数量。 由于空的区块掩码代表一个区块掩码，每个区块都会返回 true ，这个函数可能不知道区块掩码的真实大小，因此它需要一个参数来表示区块的总数量。 如果该对象已被初始化为一个非空的掩码（即，如果size()函数返回大于0的东西，或者等价地，如果respresent_the_all_selected_mask()返回false），那么该参数可以被省略，而size()的结果被取。
