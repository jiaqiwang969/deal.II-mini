Detailed Description

template<typename number>
class MeshWorker::LocalResults< number >

提供剪贴簿的类，以填充局部整合的结果。根据网格工作者循环所执行的任务，局部结果可以是不同的类型。它们可以是标量，也可以是等于积分中使用的自由度数的向量，或者是同样大小的方阵。所有这些都有一个共同点，那就是它们是在一个单元或面上进行局部积分的结果。哪种对象是一个操作的结果，是由使用它们的汇编器决定的。也是装配者决定how many每种对象的产生（例如，一个装配者可以同时创建质量和刚度矩阵的局部贡献），以及将局部结果的数组设置为所需的大小。 该类的接口允许通过以下函数访问所有这些信息。

  1. 标量：n_values()返回该类对象存储的标量数量，通过value()函数访问它们。
  2. 向量：n_vectors()返回该类对象存储的向量数量（每个向量的长度等于该单元上发生积分的自由度数量）。 向量是由vector()函数访问的。
  3. 矩阵：n_matrices()返回存储的矩阵数量，每个矩阵的维数等于每个单元的自由度数。矩阵由matrix()访问，第二个参数是false。这些矩阵在同一个单元中耦合自由度。对于跨面的通量，还有一组同样大小的矩阵，这些矩阵的维度与两个单元的自由度有关。这些矩阵可以通过matrix()访问，使用第二个参数true。

本地矩阵由 info 对象的reinit()初始化，然后由Assembler类组装成全局系统。
