Detailed Description

template<typename MatrixType, class RelaxationType, typename number>
class MGSmootherBlock< MatrixType, RelaxationType, number >

用于块状向量的通用平滑器类。该类通过初始化一个矩阵和一个平滑器对象，给了选择块状平滑器的完全自由。因此，每一级的平滑器对象必须由手工构建。

◆ apply()

void MGSmootherBase< BlockVector< number > >::apply ( const unsigned int  level,
BlockVector< number > &  u,
const BlockVector< number > &  rhs 
) const
virtualinherited

与smooth()函数相反，这个函数应用了平滑的动作，覆盖了向量u中以前的内容，这个函数必须等同于以下代码

u = 0;
smooth(level, u, rhs);

但通常可以比前者更有效地实现。如果一个特定的平滑器没有覆盖apply()方法，就会使用这里描述的默认实现。 在多网格预处理接口中，apply()方法被用于预平滑操作，因为解向量中以前的内容需要被覆盖以获得新的传入残差。另一方面，所有后续的操作都需要平滑已经存在于向量 u 中的内容，给定的右手边，这是由smooth()完成的。
