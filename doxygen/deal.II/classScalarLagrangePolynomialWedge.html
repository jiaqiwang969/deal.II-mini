Detailed Description

template<int dim>
class ScalarLagrangePolynomialWedge< dim >

定义在楔形实体上的多项式。该类是FE_WedgeP的基础。 多项式是通过一个 BarycentricPolynomials<2>::get_fe_p_basis(degree) 和一个 BarycentricPolynomials<1>::get_fe_p_basis(degree), 的张量乘积创建的，但是，为了更好地匹配FiniteElement的定义，重新进行了列举。
