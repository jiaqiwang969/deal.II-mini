Detailed Description

template<typename BaseIterator>
class FilteredIterator< BaseIterator >::PredicateBase

用于封装谓词对象的基类。由于谓词可以有不同的类型，而且我们不想把这些类型编码到过滤迭代器类的模板参数列表中，所以我们使用一个带有抽象函数的基类和模板化的派生类，通过虚拟函数实现对实际谓词类型的使用。
