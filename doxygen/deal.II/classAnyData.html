template<typename type >
const type AnyData::read ( const std::string &  name) const

Dedicated read only access by name. 对于一个常量对象，这个函数等于 entry()。对于一个非常量对象，它强制对数据进行只读访问。特别是，如果没有找到该对象或无法转换类型，它会抛出一个异常。 如果不希望出现这样的异常，请使用try_read()代替。

Warning
不要对作为指针的存储对象使用这个函数。 使用read_ptr()代替!

◆ serialize()

const unsigned int  version 
)
inlineinherited

为了使用BOOST序列化库进行序列化，将此对象的数据读入或写入一个流中。 这个函数实际上并没有对这个类的任何成员变量进行序列化。原因是这个类所存储的只是谁订阅了这个对象，但在存储这个对象的内容时，谁订阅了这个对象，与恢复时谁订阅了这个对象并不一定有关系。因此，我们不希望在恢复时覆盖订阅者，那么就没有理由在一开始就把订阅者写出来。
