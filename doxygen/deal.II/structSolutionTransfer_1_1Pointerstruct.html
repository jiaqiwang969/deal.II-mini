Detailed Description

template<int dim, typename VectorType = Vector<double>, int spacedim = dim>
struct SolutionTransfer< dim, VectorType, spacedim >::Pointerstruct

所有的单元格数据（dof指数和dof值）都应该可以从每个单元格中访问。由于每个单元只有一个 user_pointer, ，所以需要将数据的多个指针打包到一个结构中。请注意，在我们的案例中，每个单元都需要向量<unsigned int>="">索引（如果该单元将被细化）或向量<double> dof_values（如果该单元的孩子将被删除），因此一个 user_pointer应该足够了，但为了允许一些错误检查和保护用户不犯用户错误， user_pointer 将被这个结构 "乘以"。
