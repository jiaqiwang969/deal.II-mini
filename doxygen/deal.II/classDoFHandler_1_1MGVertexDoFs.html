Detailed Description

template<int dim, int spacedim = dim>
class DoFHandler< dim, spacedim >::MGVertexDoFs

一个数据结构，用于存储与特定顶点相关的DoF指数。与单元不同，顶点生活在多网格层次结构的多个层面上；因此，我们需要为每个顶点存储它所处的每个层面的DoF指数。这个类就是这样做的。
