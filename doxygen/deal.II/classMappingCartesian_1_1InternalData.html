Detailed Description

template<int dim, int spacedim = dim>
class MappingCartesian< dim, spacedim >::InternalData

存储映射的内部数据。见 Mapping::InternalDataBase 的广泛描述。 这包括创建对象时计算一次的数据（在get_data()中），以及该类希望从调用fill_fe_values()、fill_fe_face_values()或fill_fe_subface_values()之间存储的数据，直到以后可能从有限元调用转化()等函数。后一类的成员变量被标记为 "可变"。
