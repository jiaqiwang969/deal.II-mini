Detailed Description

template<int dim, typename Number = double>
class Functions::Monomial< dim, Number >

一个代表单项式的函数对象的类。单项式是只有一个单项的多项式，即在1-d中它们的形式为 \(x^\alpha\) ，在2-d中的形式为 \(x_1^{\alpha_1}x_2^{\alpha_2}\) ，而在3-d中的形式为 \(x_1^{\alpha_1}x_2^{\alpha_2}x_3^{\alpha_3}\) 。因此，单项式是由指数的 \(dim\) 元组来描述的。因此，该类的构造函数需要一个Tensor<1,dim>来描述指数的集合。大多数时候，这些指数当然是整数，但实数指数当然也同样有效。当基数是负数时，指数不可能是实数。
