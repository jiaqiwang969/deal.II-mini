Detailed Description

这类对象用于通知内部应用程序由外循环引发的变化。它们通过 Operator::notify() 被交给应用程序，如何处理它们由实际的应用程序决定。 事件被组织成一个可扩展的二进制枚举器。每个类都可以使用assign()添加自己的事件。一个典型的代码例子是

class A
{
static Event event;
};
Event A::event = Event::assign("Event for A");
