Detailed Description

template<typename number = double>
class MeshWorker::Assembler::CellsAndFaces< number >

计算一个或几个函数的单元和面的贡献，通常用于误差估计。对于一个给定的单元或面，结果被储存在哪个组件中的信息是由其user_index变量传递的。因此，在使用这个类之前，你需要确保适当地设置这些变量。

template<typename number >
void MeshWorker::Assembler::CellsAndFaces< number >::initialize ( AnyData &  results,
bool  separate_faces = true 
)
inline

初始化函数，指定 results 向量以及是否应单独收集脸部数据。 results 应该包含两个名为 "单元格 "和 "面孔 "的块向量（后者只有在 separate_faces 为真时才有）。在这两个中的每一个中，每个块都应该有相同的大小，并且大到足以容纳它所使用的循环所覆盖的单元格和面孔中设置的所有用户指数。通常，对于估计器来说，这分别是 Triangulation::n_active_cells() 和 Triangulation::n_faces(), 。 使用BlockVector似乎很麻烦，但它允许我们同时组装几个函数，每个区块都有一个。误差估计的典型情况是在每个向量中只是有一个块。
