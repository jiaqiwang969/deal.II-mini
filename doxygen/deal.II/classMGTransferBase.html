Detailed Description

template<typename VectorType>
class MGTransferBase< VectorType >

基类用于声明在多网格背景下实现向量延长和限制的具体类所需的操作。这个类是抽象的，没有这些操作的实现。 有几个派生类，反映了细格离散化和多层次实现的向量类型和编号是独立的。 如果你对一个单一的PDE或对你的完整方程组使用多网格，你将和多网格一起使用MGTransferPrebuilt。在精细网格上使用的矢量类型以及用于多级操作的矢量类型可以是Vector或BlockVector。在这两种情况下，MGTransferPrebuilt将对解决方案的所有组成部分进行操作。

限制一个从from_level层到from_level-1层的向量，并将这个限制添加到dst。如果from_level层的单元所覆盖的区域小于from_level-1层的区域（局部细化），那么dst中的一些自由度是有效的，将不会被改变。对于其他自由度，限制的结果将被添加。

  • src是一个向量，其元素数量与较细层次上的自由度相同
  • dst的元素数量与较粗层次上的自由度相同。

Implemented in MGTransferPrebuilt< VectorType >, MGTransferGlobalCoarsening< dim, VectorType >, MGTransferBlockMatrixFree< dim, Number >, MGTransferBlockSelect< number >, MGTransferSelect< number >, MGTransferBlock< number >, and MGTransferMatrixFree< dim, Number >.

为了使用BOOST序列化库进行序列化，将此对象的数据读入或写入一个流中。 这个函数实际上并没有对这个类的任何成员变量进行序列化。原因是这个类所存储的只是谁订阅了这个对象，但在存储这个对象的内容时，谁订阅了这个对象，与恢复时谁订阅了这个对象并不一定有关系。因此，我们不希望在恢复时覆盖订阅者，那么就没有理由在一开始就把订阅者写出来。
