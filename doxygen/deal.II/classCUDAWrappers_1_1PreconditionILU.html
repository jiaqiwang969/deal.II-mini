Detailed Description

template<typename Number>
class CUDAWrappers::PreconditionILU< Number >

该类为 CUDAWrappers::SparseMatrix 矩阵实现了一个不完整的LU因子化预处理程序。 该实现与cuSPARSE文档（https://docs.nvidia.com/cuda/cusparse/index.html::cusparse-lt-t-gt-csrilu02）中记载的实现非常接近。

Note
这个模板的实例化提供给 <float> 和 <double>. 。
