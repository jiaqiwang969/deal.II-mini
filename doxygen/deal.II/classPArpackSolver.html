Detailed Description

template<typename VectorType>
class PArpackSolver< VectorType >

使用PARPACK的接口。PARPACK是一个Fortran77子程序的集合，旨在解决大规模的特征值问题。这里我们为PARPACK的例程 pdneupd, pdseupd 、 pdnaupd, pdsaupd 提供接口。 该软件包被设计用来计算一般n乘n矩阵A的几个特征值和相应的特征向量。它最适用于大的稀疏矩阵A。 在这个类中，我们利用了应用于广义特征谱问题 \((A-\lambda B)x=0\) 的方法，用于 \(x\neq0\) ；其中 \(A\) 是一个系统矩阵， \(B\) 是一个质量矩阵，而 \(\lambda, x\) 分别是一组特征值和特征向量。 ArpackSolver可以通过以下方式在应用代码中使用。

SolverControl solver_control (1000, 1e-9);
const unsigned int num_arnoldi_vectors = 2*size_of_spectrum + 2;
PArpackSolver<V>::AdditionalData
additional_data(num_arnoldi_vectors,
::PArpackSolver<V>::largest_magnitude,
true);
PArpackSolver<V> eigensolver (solver_control,
mpi_communicator,
additional_data);
eigensolver.set_shift(sigma);
eigensolver.reinit(locally_owned_dofs);
eigensolver.solve (A,
B,
OP,
lambda,
x,
size_of_spectrum);

对于广义的特征值问题 \(Ax=B\lambda x\) ，其中变量 size_of_spectrum 告诉PARPACK要解决的特征向量/特征值对的数量。这里， lambda 是一个包含计算的特征值的向量， x 是一个包含计算的特征向量的 V 类型的对象的向量。 目前，只有(P)Arpack的三种模式被实现。在模式3（默认）中， OP 是对矩阵A的逆运算。

  • sigma B，其中 sigma 是一个移位值，默认设置为零。而在模式2中， OP 是 M 的逆运算。最后，模式1对应于没有谱系转换的标准特征值问题 \(Ax=\lambda x\) 。模式可以通过AdditionalData对象指定。请注意，对于移位和反转（模式=3），所寻求的特征对是应用谱系转换后的特征对。 OP 可以通过使用LinearOperator来指定。

const double shift = 5.0;
const auto op_A = linear_operator<vector_t>(A);
const auto op_B = linear_operator<vector_t>(B);
const auto op_shift = op_A
- shift op_B;
SolverControl solver_control_lin (1000, 1e-10,false,false);
SolverCG<vector_t> cg(solver_control_lin);
const auto op_shift_invert =
inverse_operator(op_shift, cg, PreconditionIdentity ());

该类旨在与MPI一起使用，可以在任意的向量和矩阵分布式类上工作。 对称和非对称的 A 都支持。 关于PARPACK例程 pdneupd 、 pdseupd, pdnaupd, pdsaupd 如何工作以及如何适当设置参数的进一步信息，请查看PARPACK手册。

◆ WhichEigenvalues

template<typename VectorType>
enum PArpackSolver::WhichEigenvalues

一个枚举，列出在solve()函数中计算哪些特征值的可能选择。注意，这与应用移位和反转（目前唯一支持的谱系变换）后的问题相对应。 一个特定的选择是基于对称或非对称矩阵 A 考虑的限制。

Enumerator
algebraically_largest 

代数上最大的特征值。

algebraically_smallest 

代数上最小的特征值。

largest_magnitude 

具有最大量级的特征值。

smallest_magnitude 

具有最小量级的特征值。

largest_real_part 

具有最大实部的特征值。

smallest_real_part 

具有最小实部的特征值。

largest_imaginary_part 

具有最大虚部的特征值。

smallest_imaginary_part 

具有最小虚部的特征值。

both_ends 

从频谱的高端计算一半的特征值，另一半从低端计算。如果要求的特征向量的数量是奇数，那么额外的特征向量来自谱的高端。
