Detailed Description

template<int dim>
class ScalarLagrangePolynomialPyramid< dim >

一个命名空间，用于提供对单数参考单元实体（即三角形和四面体）的支持的函数和类。 定义在金字塔实体上的多项式。该类是FE_PyramidP的基础。

计算 unit_point. 的多项式的值和导数，向量的大小必须是零或者等于n()。 在第一种情况下，函数将不计算这些值。 如果你需要所有多项式的值或导数，那么使用这个函数，而不是使用任何compute_value, compute_grad 或 compute_grad_grad 函数，见下文，在所有张量积多项式上循环。 。

Note
目前，只有向量 values 和 grads 被填充。
