Detailed Description

template<int spacedim>
struct internal::TriangulationImplementation::Iterators< 1, spacedim >

这个类实现了一些不同维度的类型。 这些是只针对一维情况的声明。参见 Iterators on mesh-like containers 模块以了解更多信息。 一个 line_iterator 别名为在Triangulation<1>对象的 行成员变量上操作的一个迭代器。一个 active_line_iterator只对活动线进行操作。 raw_line_iterator对象对所有的线进行操作，无论是否使用。 由于我们是在一维中，所以声明了以下身份。

using raw_cell_iterator = raw_line_iterator;
using cell_iterator = line_iterator;
using active_cell_iterator = active_line_iterator;

为了能够在Triangulation<1>中声明 begin_quad 等， quad_iterators 被声明为InvalidAccessor上的迭代器。因此，这些类型存在，但没有用，而且肯定会使任何非自愿的使用变得明显。这同样适用于六面体迭代器。 这同样适用于 face_iterator 类型，因为线条除了顶点之外没有任何子结构，不过这是以不同的方式处理的。
