Detailed Description

template<int dim, typename VectorType, class TRANSFER>
class PreconditionMG< dim, VectorType, TRANSFER >

多级预处理器。在这里，我们收集了多级预处理所需的所有信息，并为LAC迭代方法提供标准接口。 此外，它需要函数void copy_to_mg(const VectorType&)来存储多级方法右侧的 src ，void copy_from_mg(VectorType&)来存储 dst. 中的v循环结果。 如果VectorType实际上是一个块向量，并且TRANSFER对象支持为每个块使用单独的DoFHandler，这个类也允许被初始化为每个块的单独DoFHandler。
