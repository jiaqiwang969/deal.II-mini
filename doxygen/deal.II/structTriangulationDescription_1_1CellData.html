Detailed Description

template<int dim>
struct TriangulationDescription::CellData< dim >

每个本地相关单元所需的信息，存储在Description中，并在构建三角结构时使用。该结构存储了单元ID、子域ID和水平子域ID，以及与manifold_id和boundary_id相关的信息。

Note
与 CellData, 类似，该结构也存储单元格的信息。然而，与 CellData, 不同的是，它还存储一个唯一的id、分区信息以及与单元格面和边相关的信息。
