Detailed Description

template<typename Number>
class CUDAWrappers::PreconditionIC< Number >

该类实现了 对称 CUDAWrappers::SparseMatrix 矩阵的不完全Cholesky因子化（IC）预处理。 该实现与cuSPARSE文档（https://docs.nvidia.com/cuda/cusparse/index.html::cusparse-lt-t-gt-csric02）中记载的实现非常接近。

Note
本模板的实例化提供给 <float> 和 <double>. 。
