Detailed Description

template<class clock_type_>
struct Timer::ClockMeasurements< clock_type_ >

计时器类存储了两个不同时钟的计时信息：一个挂钟和一个CPU使用时钟。由于处理这两个时钟的逻辑在大多数地方都是相同的，所以我们把每个时钟的相关测量数据收集到这个 struct 。

Template Parameters
clock_type_其测量值被存储的时钟的类型。这个类应该符合 std::chrono 所期望的通常的时钟接口（即正确的别名和一个静态 now() 方法）。
