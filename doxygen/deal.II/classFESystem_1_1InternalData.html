template<int dim, int spacedim = dim>
class FESystem< dim, spacedim >::InternalData

std::vector< std::unique_ptr<typename FiniteElement<dim, spacedim>::InternalDataBase> > FESystem< dim, spacedim >::InternalData::base_fe_datas
private

指向每个基元的 InternalData 对象的指针。它们被 set_ 和 get_fe_data 函数所访问。 这个向量的大小由InternalData构造函数设置为 n_base_elements 。 它由 get_data 函数填充。 请注意，由于基类的每个实例的数据必然是相同的，我们只需要有多少个基类元素就有多少个这样的对象，而不考虑它们的多重性。
