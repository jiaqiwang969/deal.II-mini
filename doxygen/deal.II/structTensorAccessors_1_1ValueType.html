Detailed Description

template<typename T>
struct TensorAccessors::ValueType< T >

这个类提供了一个本地别名 value_type ，表示用operator[](unsigned int)访问的结果类型。更准确地说， value_type将是

  • T::value_type 如果T是一个提供别名的张量类 value_type ，并且没有const限定词。
  • const T::value_type 如果T是一个提供别名的张量类 value_type ，并且确实有一个const限定词。
  • const T::value_type 如果T是一个提供别名的张量类 value_type ，并且确实有一个const限定词。
  • A if T is of array type A[...]
  • const A if T is of array type A[...] 并且确实有一个const限定词。
