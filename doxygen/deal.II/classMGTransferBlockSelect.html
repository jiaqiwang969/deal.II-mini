Detailed Description

template<typename number>
class MGTransferBlockSelect< number >

MGTransferBase接口的实现，用于块状矩阵和简单向量。该类使用MGTransferBlockBase选择单一块。对Vector对象实现了网格间转移操作，对Vector和BlockVector实现了常规和多网格向量间的复制功能。 请参阅MGTransferBase来了解哪一个转移类最适合你的需要。
