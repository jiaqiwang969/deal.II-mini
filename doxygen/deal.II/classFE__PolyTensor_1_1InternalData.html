Detailed Description

template<int dim, int spacedim = dim>
class FE_PolyTensor< dim, spacedim >::InternalData

FE_PolyTensor的独立于细胞的数据字段。在参考单元格上存储形状函数的值和它们的导数，以便以后使用。 所有表格的组织方式是，正交点k的形状函数i的值可以通过索引<i>(i,k)来访问。
