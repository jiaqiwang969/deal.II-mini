Detailed Description

template<int dim, int n_rows, int n_columns, typename Number, typename Number2>
struct internal::EvaluatorTensorProduct< evaluate_symmetric, dim, n_rows, n_columns, Number, Number2 >

使用基函数的张量积形式的1d-3d形状函数的内部评估器。该类专门针对 "对称 "有限元的基于张量积的元素的一般应用，即当形状函数关于0.5的对称性和正交点也是如此。

Template Parameters
dim应用该类的空间维度
n_rows变换矩阵中的行数，对应于通常张量收缩设置中的1d形状函数的数量
n_columns变换矩阵中的列数，对应于通常张量收缩设置中的1d形状函数的数量
]Number 用于输入和输出数组的抽象数字类型
Number2用于系数数组的抽象数字类型（默认为与输入/输出数组相同的类型）；必须用Number实现操作符*，并产生Number作为输出，才能成为有效类型。
