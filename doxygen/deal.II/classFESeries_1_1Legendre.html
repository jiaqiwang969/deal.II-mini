Detailed Description

template<int dim, int spacedim = dim>
class FESeries::Legendre< dim, spacedim >

一个用于计算标量FE（或矢量FE的单个分量）场在参考元素上扩展为一系列Legendre函数的类。 Legendre函数是Legendre微分方程

\[ \frac{d}{dx}\left([1-x^2] \frac{d}{dx} P_n(x)\right) + n[n+1] P_n(x) = 0 \]

的解，可以用Rodrigues公式

\[ P_n(x) = \frac{1}{2^n n!} \frac{d^n}{dx^n}[x^2-1]^n. \]

来表达，这些多项式相对于区间 \( L^2 \)

\[ \int_{-1}^1 P_m(x) P_n(x) = \frac{2}{2n + 1} \delta_{mn} \]

的内积是正交的，是完整的。 在 \( L^2 \) 上的 \( [0;1] \) 的正交多项式家族可以通过

\[ \widetilde P_m = \sqrt{2} P_m(2x-1). \]

构建。 参考元素 \( [0;1] \) 上的任意标量FE场可以在完全正交基础上展开为

\[ u(x) = \sum_{m} c_m \widetilde P_{m}(x). \]

]从基的正交特性可以看出，

\[ c_m = \frac{2m+1}{2} \int_0^1 u(x) \widetilde P_m(x) dx . \]

该类计算系数 \( c_{\bf k} \) 使用 \( dim \) -维Legendre多项式，使用张量积规则从 \( \widetilde P_m(x) \) 构建。

构造函数，初始化所有需要的数据结构。 n_coefficients_per_direction 定义了每个方向的系数数， fe_collection 是将用于展开的 hp::FECollection ， q_collection 是用于整合 fe_collection. 中每个FiniteElement的展开的 hp::QCollection 由于Legendre展开只能在标量场上进行，这个类不会对矢量值的有限元操作，因此会抛出一个断言。然而，有限元场的每个分量可以分别被视为标量场，对其进行Legendre扩展也是可能的。为此，可选的参数 component 定义了每个有限元的哪个分量将被使用。 默认值 component 只适用于标量FEs，在这种情况下，它表示唯一的分量将被分解。对于矢量FE，必须明确提供一个非默认值。

◆ precalculate_all_transformation_matrices()

template<int dim, int spacedim>
void FESeries::Legendre< dim, spacedim >::precalculate_all_transformation_matrices ( )

计算所有的转换矩阵，将有限元解转移到系列展开表示。 这些矩阵将通过调用calculate()按需生成，并存储起来供重复使用。通常情况下，这个操作会消耗大量的工作量。有了这个函数，所有的矩阵将被提前计算。 这样，我们就可以将其昂贵的生成与实际应用分开。
