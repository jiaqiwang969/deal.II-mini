Detailed Description

template<typename Number>
class BlockVector< Number >

一个基于deal.II向量的块向量的实现。虽然基类提供了大部分的接口，但这个类处理了向量的实际分配，并提供了底层向量类型的特定函数。

Note
这个模板的实例提供给 <float> 和 <double>; 其他人可以在应用程序中生成（见手册中的 Template instantiations 部分）。

把这个向量的内容和另一个向量v交换。我们可以通过一个临时变量和复制数据元素来完成这个操作，但是这个函数的效率明显更高，因为它只交换了两个向量的数据指针，因此不需要分配临时存储空间和移动数据。 这个函数类似于所有C++标准容器的swap()函数。此外，还有一个全局函数swap(u,v)，它简单地调用u.swap(v)，同样与标准函数相类似。

◆ locally_owned_elements()

IndexSet BlockVectorBase< Vector< Number > >::locally_owned_elements ( ) const
inherited

返回一个索引集，描述这个向量的哪些元素是由当前处理器拥有的。请注意，这个索引集不包括这个向量可能作为鬼魂元素存储在本地，但实际上是由另一个处理器拥有的元素。因此，如果这是一个分布式向量，在不同处理器上返回的索引集将形成不相交的集合，加起来就是完整的索引集。很明显，如果一个向量只在一个处理器上创建，那么结果将满足

vec.locally_owned_elements() == complete_index_set (vec.size())

对于块状向量，这个函数返回各个块的本地拥有的元素的并集，以它们各自的索引偏移量进行移位。
