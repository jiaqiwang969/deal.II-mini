Detailed Description

template<int N>
class TableIndices< N >

一个代表固定大小的索引阵列的类。 它被用于像TableBase和SymmetricTensor类这样的张量对象中，表示指数的嵌套选择。

Template Parameters
N每个对象中存储的指数的数量。
