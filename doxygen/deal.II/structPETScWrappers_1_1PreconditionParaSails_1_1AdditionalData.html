◆ symmetric

unsigned int PETScWrappers::PreconditionParaSails::AdditionalData::symmetric

这个参数指定了要解决的问题的类型。

  • 0: 非对称和/或不确定问题，以及非对称预处理
  • 1: SPD问题，以及SPD（因子）预处理
  • 2: 非对称、确定问题，以及SPD（因子）预处理

默认为symmetric = 1。

◆ threshold

double PETScWrappers::PreconditionParaSails::AdditionalData::threshold

稀疏化是通过丢弃幅度小于thresh的非零点来实现的。较低的thresh值会导致更精确，但也更昂贵的预处理程序。 默认值是thresh = 0.1。设置thresh < 0会自动选择一个阈值，这样thresh就代表了被放弃的非零元素的比例。例如，如果thresh =

  • .9，那么B将包含给定矩阵A中约百分之十的非零元素。

◆ filter

double PETScWrappers::PreconditionParaSails::AdditionalData::filter

过滤是一个后处理过程，filter代表在创建近似的反稀疏模式后被丢弃的非零元素的一部分。默认值是filter = 0.05。设置filter < 0会自动选择一个值，这样-filter代表被丢弃的非零元素的比例。例如，如果thresh =

  • .9，那么在计算的近似逆中约有90的条目被放弃。
