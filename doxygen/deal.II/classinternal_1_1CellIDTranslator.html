Detailed Description

template<int dim>
class internal::CellIDTranslator< dim >

A class that helps to translate between CellIDs and globally uniquely indices. The resulting index space might be non-contiguous in the case of locally refined meshes.

The following code snippet how to set up an IndexSet object for active cells, using this class:

// set up translator
CellIDTranslator<dim> translator(tria.n_global_coarse_cells(),
tria.n_global_levels());
// initialize the index set with the correct size
IndexSet index_set(translator.size());
// convert IDs to indices and add them to the index set
for(const auto & cell : tria.active_cell_iterators())
index_set.add_index(translator.translate(cell));
Note
The definition of the indices are not relevant.
