template<int dim, int spacedim>
std::vector<std::shared_ptr<const Triangulation<dim, spacedim> > > MGTransferGlobalCoarseningTools::create_geometric_coarsening_sequence ( const Triangulation< dim, spacedim > &  tria)

对于一个给定的三角形 tria, 通过对所提供的三角形重复进行全局粗化来确定几何粗化序列。

Note
为方便起见，在返回向量的最后一项中存储了对输入三角形的引用。
目前，没有为 parallel::fullydistributed::Triangulation. 实现。
返回三角形的类型与输入三角形的类型相同。

template<int dim, int spacedim>

const RepartitioningPolicyTools::Base< dim, spacedim > &  policy,
const bool  preserve_fine_triangulation,
const bool  repartition_fine_triangulation 
)

类似于上面的函数，但也取一个 policy ，用于在更粗的层次上重新划分三角形。如果 preserve_fine_triangulation 被设置，输入的三角形不被改变，否则三角形被粗化。如果 repartition_fine_triangulation 被设置，最细层的三角形也会被重新划分。如果标志被设置为true/false，则输入的三角图被简单地用作最细的三角图。

Note
为方便起见，在返回向量的最后一项中存储了对输入三角形的引用。
返回三角形的类型是 parallel::fullydistributed::Triangulation. 。
目前，只对 parallel::distributed::Triangulation. 实现。
