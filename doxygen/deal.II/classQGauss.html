Detailed Description

template<int dim>
class QGauss< dim >

用于数值积分的Gauss-Legendre系列的正交规则。 这些正交规则的系数是由Numerical Recipes中描述的函数计算的。

◆ get_tensor_basis()

template<int dim>
std::conditional< dim==1, std::array< Quadrature< 1 >, dim >, const std::array< Quadrature< 1 >, dim > & >::type Quadrature< dim >::get_tensor_basis ( ) const
inherited

如果正交公式是张量积，此函数返回 dim 一维基础对象。 否则，不允许调用此函数。 对于 dim 等于1的情况，我们不能将 std::array 作为常量引用返回，必须通过值来返回。在这种情况下，数组将总是包含一个元素（ this). ）。

Note
这个函数的实际返回类型是
std::conditional<dim == 1,
std::array<Quadrature<1>, dim>,
const std::array<Quadrature<1>, dim> &>::type
在线文档中对该类型进行了缩写，以提高本页面的可读性。
