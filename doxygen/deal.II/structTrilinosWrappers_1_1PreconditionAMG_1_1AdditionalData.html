Detailed Description

一个数据结构，用于控制代数多重网格的设置细节。这里详述的标志会被传递给Trilinos ML实现。一个当前类型的结构被传递给PreconditionAMG的构造函数。

Note
设置参数反映了这个对象中的当前设置，各种选项既可以通过成员变量的状态直接设置（例如 "平滑器：类型"），也可以间接设置（例如 "聚合：类型"）。如果你希望对AMG预处理程序的配置进行精细控制，那么你可以使用这个函数创建参数列表（方便地设置运算符的空位），改变相关设置，并使用修正后的参数列表作为 PreconditionAMG::initialize(), 的参数，而不是AdditionalData对象本身。 关于有哪些选项可供修改，请参见Trilinos ML package的文档。

◆ smoother_sweeps

unsigned int TrilinosWrappers::PreconditionAMG::AdditionalData::smoother_sweeps

决定应该进行多少次平滑器的扫频。当标志elliptic被设置为true，即对于椭圆或几乎椭圆的问题，切比雪夫平滑器的多项式程度被设置为smoother_sweeps。扫频指的是在切比雪夫情况下进行的矩阵-向量乘积的数量。在非椭圆情况下，smoother_sweeps设置后平滑的SSOR松弛扫频次数。
