Detailed Description

template<typename Number>
class BlockSparseMatrixEZ< Number >

一个由SparseMatrixEZ类型的块组成的块矩阵。 与其他块对象一样，这个矩阵可以像SparseMatrixEZ一样使用，当涉及到对条目的访问时。然后，还有一些函数用于与BlockVector相乘，以及对单个块的访问。
