template<typename Iterator , typename MemorySpaceType = MemorySpace::Host>
ArrayView< typename std::remove_reference< typename std::iterator_traits< Iterator >::reference >::type, MemorySpaceType > make_array_view ( const Iterator  begin,
const Iterator  end 
)

创建一个ArrayView，它需要一对迭代器作为参数。ArrayView的类型是从迭代器的值类型推断出来的（例如，从两个常数迭代器创建的视图将有一个常数类型）。

Warning
迭代器 begin 和 end 必须绑定（以通常的半开放方式）一个连续的内存范围内的值。这个函数的目的是用于像 boost::container::small_vector or std::vector 这样的容器中的迭代器，而不会在例如 boost::container::stable_vector or std::deque 中正确工作。在调试模式下，我们检查所提供的迭代器是否确实代表连续的内存。
