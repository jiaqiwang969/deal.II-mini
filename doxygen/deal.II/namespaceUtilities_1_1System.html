◆ posix_memalign()

void Utilities::System::posix_memalign ( void **  memptr,
std::size_t  alignment,
std::size_t  size 
)

调用系统函数posix_memalign，如果没有，则调用一个替代函数，以分配具有某种最小对齐方式的内存。然后第一个参数将返回这个内存块的指针，以后可以通过标准的 free 调用来释放。

Parameters
memptr一个指针变量的地址，在这个调用之后将指向所分配的内存。
alignment内存块的最小对齐方式，字节数。
size要分配的内存块的大小，以字节为单位。
Note
这个函数在内部检查错误代码，而不是把这个任务留给调用站点。
