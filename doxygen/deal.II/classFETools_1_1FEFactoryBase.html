Detailed Description

template<int dim, int spacedim = dim>
class FETools::FEFactoryBase< dim, spacedim >

一个用于创建给定程度的有限元素的工厂对象的基类。每当人们想用一种透明的方式来创建有限元对象时，就会调用派生类。 该类在 FETools::get_fe_by_name() 和 FETools::add_fe_name() 函数中使用。

template<class Archive >
void Subscriptor::serialize ( Archive &  ar,
const unsigned int  version 
)
inlineinherited

为了使用BOOST序列化库进行序列化，将此对象的数据读入或写入一个流中。 这个函数实际上并没有对这个类的任何成员变量进行序列化。原因是这个类所存储的只是谁订阅了这个对象，但在存储这个对象的内容时，谁订阅了这个对象，与恢复时谁订阅了这个对象并不一定有关系。因此，我们不希望在恢复时覆盖订阅者，那么就没有理由在一开始就把订阅者写出来。
