◆ value()

template<int dim, int spacedim = dim>
value_type FEInterfaceViews::Vector< dim, spacedim >::value ( const bool  here_or_there,
const unsigned int  interface_dof_index,
const unsigned int  q_point 
) const

返回该视图选择的具有界面dof索引 interface_dof_index 的向量分量在正交点 q_point. 的值 参数 here_or_there 在该正交点的法向量方向定义的上游值和下游值之间选择。如果 here_or_there 为真，则使用界面的第一个单元的形状函数。 换句话说，当从界面的两个单元之一接近给定正交点时，该函数返回形状函数的极限值。

Note
这个函数通常用于根据一个方向来挑选上游或下游的值。这可以通过使用 (direction normal)>0 作为该函数的第一个参数来实现。
