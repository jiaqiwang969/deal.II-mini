Detailed Description

template<typename MatrixType, class RelaxationType, typename VectorType>
class MGSmootherRelaxation< MatrixType, RelaxationType, VectorType >

使用满足 放松概念 的求解器进行平滑处理。 该类在每个层次上执行平滑操作。该操作可以由几个参数控制。首先，松弛参数 omega 被用于底层松弛方法中。 steps 是最细层次上的松弛步数（如果 variable 关闭，则在所有层次上）。如果 variable 是 true, ，则每一个较粗的层次上的平滑步数是双倍的。这导致了一种具有W-循环复杂性的方法，但节省了网格转移。这是Bramble at al.提出的方法。 选项 symmetric 按照Bramble的建议，在每个步骤中交替使用平滑器和其转置。 transpose 使用Tstep的转置平滑操作，而不是放松方案的常规step。 如果你使用块矩阵，第二个 initialize 函数提供了提取单个块进行平滑的可能性。在这种情况下，多棱镜方法必须只用于与该单块相关的向量。 该库包含SparseMatrix<.>和Vector<.>的实例化，其中模板参数是 float和 double. 的所有组合，其他实例化可以通过包括文件mg_smoother.templates.h来创建。

) const

平滑化的应用变体，在调用平滑函数之前将向量u设置为零。这个函数等同于以下代码

u = 0;
smooth(level, u, rhs);

在多网格预处理接口中，apply()方法用于预平滑操作，因为解向量中以前的内容需要为新进来的残差所覆盖。另一方面，所有后续的操作都需要平滑已经存在于向量 u 中的内容，给定的右手边，这是由smooth()完成。

Reimplemented from MGSmootherBase< VectorType >.
