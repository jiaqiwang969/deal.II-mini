Detailed Description

template<typename MatrixType>
class MGMatrixBlockVector< MatrixType >

一个MGLevelObject<MatrixBlock>的向量，使用共享指针来实现，以便于复制和重新排列。每个矩阵块都可以通过名称来识别。

清除对象。 因为通常只需要清除单个矩阵，而不需要清除块本身，所以有一个可选的参数。如果缺少这个参数或者 false, ，所有的矩阵将被清空，但是这个对象的大小和块的位置将不会改变。如果 really_clean是 true, ，那么该对象在最后将不包含任何块。

为了使用BOOST序列化库进行序列化，将此对象的数据读入或写入一个流中。 这个函数实际上并没有对这个类的任何成员变量进行序列化。原因是这个类所存储的只是谁订阅了这个对象，但在存储这个对象的内容时，谁订阅了这个对象，与恢复时谁订阅了这个对象并不一定有关系。因此，我们不希望在恢复时覆盖订阅者，那么就没有理由在一开始就把订阅者写出来。
