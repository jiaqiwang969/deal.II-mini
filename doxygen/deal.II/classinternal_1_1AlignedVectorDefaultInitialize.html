Detailed Description

template<typename T, bool initialize_memory>
class internal::AlignedVectorDefaultInitialize< T, initialize_memory >

像AlignedVectorInitialize一样，但使用默认构造的对象作为初始化器。

Template Parameters
initialize_memory设置set命令是否应该初始化内存（通过调用拷贝构造器），或者宁可使用拷贝赋值操作符。模板对于选择适当的操作是必要的，因为有些类可能只定义这两种操作中的一种。
