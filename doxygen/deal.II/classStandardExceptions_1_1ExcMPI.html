Detailed Description

MPI错误的异常。这个异常只有在 deal.II 编译时支持MPI时才会被定义。这个异常应该和 AssertThrow 一起使用，以检查MPI函数的错误代码。比如说。

const int ierr = MPI_Isend(...);
AssertThrow(ierr == MPI_SUCCESS, ExcMPI(ierr));

或者，使用方便的宏 AssertThrowMPI 。

const int ierr = MPI_Irecv(...);
AssertThrowMPI(ierr);

如果断言失败，那么错误代码将被用于利用 MPI_Error_string 函数向屏幕打印一个有用的信息。
