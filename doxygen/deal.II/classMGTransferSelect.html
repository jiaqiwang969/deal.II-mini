Detailed Description

template<typename number>
class MGTransferSelect< number >

MGTransferBase接口的实现，用于块状矩阵和简单向量。这个类使用MGTransferComponentBase选择一个单独的组件或将几个组件分组到一个块中。转移操作符本身是为Vector和BlockVector对象实现的。 请参阅MGTransferBase来了解哪一个转移类最适合你的需要。

实际上是为分组的组件建立延长矩阵。 这个函数是MGTransferComponentBase中相同函数的前端。

  • 选定 要从多级向量复制到全局向量的块的编号。这个数字是指通过target_component重新编号。
  • mg_selected 应建立转移矩阵的块的编号。 如果mg_target_component存在，这指的是重新编号的组件。
  • target_component这个参数允许对细级向量中的组件进行分组和重新编号（见 DoFRenumbering::component_wise).
  • mg_target_component这个参数允许对级向量中的组件进行分组和重新编号（见 DoFRenumbering::component_wise). 它也会影响selected参数的行为
  • boundary_indices持有每一级的边界指数。
