Detailed Description

template<class ACCESSOR>
class MatrixIterator< ACCESSOR >

用于常数和非常数矩阵的迭代器。 这个迭代器是从实际的矩阵类型中抽象出来的，可以用于任何具有所需ACCESSOR类型的矩阵。
