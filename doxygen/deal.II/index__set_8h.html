IndexSet complete_index_set ( const IndexSet::size_type  N)
inline

创建并返回一个大小为 \(N\) 的索引集，其中包含这个范围内的每一个索引。从本质上讲，这个函数返回一个由以下方式创建的索引集

IndexSet is (N);
is.add_range(0, N);

这个函数的存在是为了让人们能够在一个步骤中创建和初始化完整的索引集，或者说是为了让人们能够写出这样的代码
